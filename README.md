This repo contains our work for the RSNA intercranial aneurysm detection competition held on kaggle.

To replicate the training of these models, follow the steps listed below:
- Select the create a new notebook option on Kaggle
- Upload these notebooks
- Select the "Add input" option and attach the RSNA Intracranial aneurysm detection 2025 Competetion Dataset.
- Run all cells in order
